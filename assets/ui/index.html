Access Dependency Analyzer

複数の Access ファイル（.accdb / .mdb）を解析し、テーブル・リンクテーブル・クエリ・依存関係を output/analysis/ フォルダへ出力します。PostgreSQL 移行計画のたたき台として利用できます。

Accessファイルをドラッグ＆ドロップ または下の「ファイル選択」ボタンから複数ファイルを指定
